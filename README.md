# HistorialEnfermedades - Prueba web móvil

Esta es una aplicación móvil Android que se encarga de llevar un historial de enfermedades. Permite que el user inicie sesión, agregue registros de enfermedades y visualize un listado de los registros que ya haya almacenado.

## Características Principales

- **Inicio de Sesión**: La autenticación de usuario se lleva a cabo con credenciales predefinidas.
- **Listado de Registros**: Muestra todos los registros capturados, ordenados por fecha de registro.
- **Búsqueda**: Filtra registros por coincidencias en los campos de paciente, doctor o malestar.
- **Registro de Enfermedades**: Permite capturar y guardar nuevos registros con detalles como fecha, paciente, doctor, teléfono, malestar e imagen.
- **Persistencia de Datos**: Los datos se almacenan localmente en el dispositivo.

## Requisitos del Sistema

- **Android Studio**: Versión 4.0 o superior.
- **JDK**: Versión 8 o superior.
- **Dispositivo Android**: Con Android 5.0 (Lollipop) o superior, o un emulador configurado.
